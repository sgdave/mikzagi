## User Manual
### folders
- 0_Inbox  
  Where raw, messy, untagged notes live. During vacation, everything goes here first.
- 1_Processing
Temporary workspace for sorting what comes from Inbox → Areas, Projects, Resources
- 2_Areas
Your key life pillars.
Each is ongoing and never “done.”
- 3_Projects
Short-term actions with an end point e.g., “Set up full home Wi-Fi upgrades”
- 4_Resources
Long-term reference material (guides, research, templates, quotes, manuals)
- 5_Reviews
Long-term reference material (guides, research, templates, quotes, manuals)
